# 🔐 OWASP API Top 10 2023 - Vulnerable API Security Testing

A hands-on **educational security project** demonstrating real-world API vulnerabilities based on the **OWASP API Security Top 10 (2023)** using Python Flask.

⚠️ **Disclaimer:**  
This project is intentionally vulnerable and designed only for learning and security testing purposes. Do NOT use in production systems.

---

## 🎯 Purpose of This Project

This project helps learners understand:

- How API vulnerabilities occur in real systems
- How attackers exploit insecure APIs
- How Broken Authentication and BOLA work in practice
- How to test APIs using Postman and Swagger
- How to identify security flaws mapped to OWASP API Top 10

---

## ⚙️ Tech Stack

- Python 3.x
- Flask
- Flask-HTTPAuth
- Flasgger (Swagger UI)
- Werkzeug Security
- Postman (for testing)
